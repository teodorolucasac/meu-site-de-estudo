O que é markdown?

Markdown é uma linguagem de marcação, como se fosse um HTML, porém com a intenção de ser mais simples.

No Markdown você consegue ler o código fonte com mais facilidade separando o que é texto e o que é marcação. Além disso markdown é facilmente convertido em HTML.

Onde usar formatação markdown?

  1. No arquivo README.md, o arquivo criado sempre que criamos um novo repositório.
  2. Nas issues, para cria-las ou comentá-las.
  3. E nas pull requests

Guia de linguagem markdown

Marcação
# Título - título de nível 1
## Título - título de nível 2
### Título - título de nível 3
--- - cria uma linha
*** - cria uma linha mais forte
~~texto~~ - tachado, riscado ao meio
**texto** - negrito
__texto__ - negrito
*texto* - itálico
> - cria uma sitação

Podemos misturar as configurações
__*texto*__ - negrio e itálico juntos

Listas
1. texto - lista ordenada
1. texto - (3 espaços) subitem de lista ordenada

* texto - lista não ordenada
* texto - subitem de lista não ordenada

- [] texto - lista de tarefas desmarcada
- [x] texto - lista de tarefas marcada

Tabela
Num | Nome | Nota
---|---|---
1 | Lucas | 10,0
2 | José | 9,0
3 | Maria | 9,5
