Navegar / Criar

cd pastaNome = avança uma a frente (pastaNome)
cd .. = volta uma pasta atrás
C:\Program Files\Git = avança varias pastas do caminho
dir = lista todas as pastas do local
mkdir NomePasta = cria uma pasta (NomePasta)
touch NomeArquivo.js = cria um arquivo
touch NomePasta/NomeArquivo2.js = cria arquivo dentro de uma pasta específica
mv NomeDoArquivo /NomeDaPasta = move um arquivo para dentro da pasta mencionada


Remover

rm NomeArquivo = remove um arquivo (NomeArquivo)
rm -r NomePasta = remove uma pasta (NomePasta)
rm -rf NomePasta .git = remove uma diretorio dentro da pasta "NomePasta"
rm -rf NomePasta = remove a pasta "NomePasta" junto com diretorio git dentro dela
rmdir NomePasta = remove diretorios vazios
=

Salvar local e remoto 1ª vez:

git init = inicia/cria um repositório remoto na pasta atual
git remote add origin https://github.com/teodorolucasac/Github-Nelio-Alves.git = prepara para subir no repositório remoto
git pull origin main = descer arquivos do remoto para local (gitignore / readme)
git add . (ou git add -A) = joga TODOS os arquivos na area de Stage preparando para salvar
*git add -u = da update em um arquivo que já está no Stage
*git add index.html = joga apenas o arquivo html no stage
git commit -m "Nome explicativo do Commit" = cria um commit
git branch -M main = informa o nome da branch no commit
git push -u origin main = faz o push para o repositório remoto

Salvar local e remoto 2ª vez:

git add .
git commit -m "Nome explicativo do Commit"
git push

Apagar commit mantendo as alterações

git reset --soft HEAD~1 = apaga commit sem retirar as alterações

Apagar commit apagando as alterações

git reset --hard códigoDoCommit = apaga o commit e também apaga as alterações a partir do codigo do commit informado

Pull (inverso do push)

git pull origin main = git pull NomeDoRemote NomeDaBranch

Pull/Push DESCARTANDO alteração do Github que não está no repositório remoto

git pull -f origin main = git pull NomeDoRemote NomeDaBranch (trás da núvem destruindo commits posteriores local)
git push -f = manda para núvem destruindo commits posteriores da núvem

Clonar projetos

git clone https://github.com/teodorolucasac/Github-Nelio-Alves.git

Apontar um repositorio remoto para outro repositorio remoto

git remote set-url origin URLdoRepositorioRemoto
git push -u origin main

git checkout código do commit = volta para um commit anterior do código informado

git checkout HEAD~2 = volta para 2 commits anteriores do main
git reset = retira as modificações do Stage
git restore --staged nomeDoArquivo.txt ou c:/caminhoda/pasta01 = retira um arquivo ou pasta específica do stage git clean -df = limpa alguma modificação na branch 01/02
git checkout -- . = limpa alguma modificação na branch 02/02
git checkout main = muda da branch master (inicial do git) para main (padrão github)
