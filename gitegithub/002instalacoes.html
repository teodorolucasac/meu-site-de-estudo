Ferramentas:

Para programar nossos códigos vamos precisar de 4 ferramentas:

Google Chrome: O Chrome geralmente vem com uma versão menor, padrão, mas neste link iremos baixar uma versão mais completa, a versão Standalone.

Visual Studio Code: O VSC tem duas versões, a Stable e a Insiders. A versão Insiders é mais atual porém ainda em teste de novos recursos. Portanto vamos baixar a Stable que dificilmente veremos bugs de testes.

Git SCM: No Git iremos baixar a versão Lateste source Release, a mais recente para Windows.

GitHub Desktop: Aqui não tem segredo, só temos a única versão disponível para download.
