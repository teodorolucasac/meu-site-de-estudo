Estudo sobre Listas

As listas são um exemplo de tags que não precisam ser fechadas, mas se fechar não tem problema.

As Listas são formadas pelas tags: "UL", "OL" e "LI"

ul: cria listas sem ordem (unordered list)

ol: cria listas em ordem (ordered list)

li: cria os itens da lista

Listas Não Ordenadas:

  • Arroz
  • Feijão
  • Macarrão

Configuração das Listas Ordenadas:

As listas não ordenadas tem outras opções de marcação, basta usar as tags "DISC", "CIRCLE" e "SQUARE". Exemplo:

  • Arroz
  • Feijão
  • Macarrão
  • Arroz
  • Feijão
  • Macarrão
  • Arroz
  • Feijão
  • Macarrão

Listas Ordenadas:

  1. Arroz
  2. Feijão
  3. Macarrão

Configuração das Listas Ordenadas:

Nas listas ordenadas, é possível começar a lista com um número diferente de 1 usando a tag "START". Exemplo:

  1. Arroz
  2. Feijão
  3. Macarrão

Também existem outras opções de lista que não seja númerica, basta usar as tags "A", "a" ou "I". Exemplo:

  1. Arroz
  2. Feijão
  3. Macarrão
  1. Arroz
  2. Feijão
  3. Macarrão
  1. Arroz
  2. Feijão
  3. Macarrão

Listas de Definição:

A lista de definição é utilizada para fazer listas como um dicionário por exemplo, teremos um termo e em baixo a definição dele.

A lista de definição é representada pelas tags "DL", "DT" e "DD".

dl: definition list, é a tag geral para a máquina entender que ali dentro terá uma lista de definição.

dt: termo, aqui fica o item que terá uma descrição.

dd: descrição do termo informado anteriormente.

Exemplo:

HTML
Linguagem de marcação para a criação de conteúdo de um site.
CSS
Linguagem de marcação para a criação do design de um site.
JavaScrip
Linguagem de programação para a criação de interatividade de um site.
