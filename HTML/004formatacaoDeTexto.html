Tags:

As tags no HTML servem para inserir conteúdos no documento como como imagens, vídeos, textos, título etc... Também passam a informação ao browser o que significa cada elemento ou conteúdo no documento.

Dão sentido semântico ao conteúdo, dando sentido ao conteúdo viabilizando uma leitura de tela à deficientes visuais por exemplo e leitores de código.

Por fim é através das marcações de tags que conseguimos interagir com o conteúdo usando CSS e JS.
