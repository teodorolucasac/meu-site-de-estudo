Aprendendo sobre Links

os links são feitos pela Tag <a href="insiralink.com">texto clicável.</a>

Link absoluto:

os links absolutos são links que direcionam para outra página. São feitos pela Tag <a href="https://www.google.com/">ir para o Google.</a>

ir para o Google (mesma aba).

Para melhorar a experiência do usuários com links absolutos colocaremos a tag "TARGET:_BLANK", assim abriremos uma nova aba ao clicar no link.

<a href="https://www.google.com/" target="_blank">ir para o Google.</a>

ir para o Google (nova aba).

Link Relativo:

Links relativos são links que direcionam o usuário para navegar na mesma página.

Utilizam a mesma TAG "A", porém utiliza o "TARGET="_SELF"". Exemplo de como voltar à página anterior:

<a href="paginaDeDestino.html" target="_self">Voltar</a>

Voltar

Link Relativo em Pastas diferentes:

Lembrando que se tiver em uma pasta adiante deve coloca o caminho corerto. Exemplo: <a href="pasta/paginaDeDestino.html" target="_self">Voltar</a>

Ou se tiver em uma 2 pastas anteriores devem voltar. Exemplo: <a href="../../paginaDeDestino.html" target="_self">Voltar</a>
