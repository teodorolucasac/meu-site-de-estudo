Estudo sobre Formulários

Os formulário começam com a tag <form></form>. Essa é a tag geral dos formulários, ela que organiza e informa que ali dentro contém um formulário.

Atributos:

type="": o atributo type sempre está presente nos formulários e diz qual é o tipo de formulário/preenchimento terá.
name="": serve para dizer que as questões são do mesmo conjunto e evitar a marcação de mais de 1 opção.
id="": define um id.
value="": define um valor.
placeholder="": escreve uma marca dágua na box.
selected: inicia habilitado nesta opção
disabled: desabilita a seleção desta opção
required: coloca a opção como obrigatório

Labels

label são tags que definem um título ao input do formulário, e usa o id do input para unir ao título. Neste caso, se eu clicar no título já é como se eu clicasse na box de texto. Exemplo:

<label for="campoNome">Nome:</label>
<input type="text" placeholder="Digite seu nome aqui" name="" id="campoNome">

Tag <input type="text">: usada para informar textos pequenos.



Tag <input type="email">: usada para informar emails. A tag específica vem com algumas pré configurações como: O campo obrigatóriamente deve conter um "@" e um "."



Tag <input type="password">: usada para informar senhas. Pré configurada para não mostrar os caracteres.



Tag <input type="radio">: usada para criar questão de multipla escolha.
Atributos mais usados: (name="", id="" e value="")

Tag <input type="checkbox">: usada para criar caixas de marcação, onde o usuário selecionará quantas quiser.
Igual o type radio, os atributos mais usados são: (name="", id="" e value="")
