Estrutura Básica da Página

Sempre começaremos nosso documento html com uma estrutura básica, contendo algumas configurações iniciais para o navegador ler melhor nossos códigos. Principalmente a parte de cima do código antes da tag </head>
Abaixo veremos o exemplo de uma estrutura mais detalhada:

<!DOCTYPE html>
<html lang="pt-br">

<head>
<meta charset="UTF-8">
<meta http-equiv="X-UA-Compatible" content="IE=edge">
<meta name="viewport" content="width=device-width, initial-scale=1.0">

<meta name="description" content="Esse é meu primeiro site!">
<meta name="keywords" content="html,css,js,javascript,estudo">
<meta name="author" content="Teodoro Lucas">

<title>Meu Site de Estudos</title>
<link rel="stylesheet" href="../style.css">
<script src="../script.js"></script>
</head>

<header>
<h1>Título</h1>
<p>Descrição</p>
</header>

<nav>
</nav>
<a href="001HTML.html">Voltar</a>

<body>
</body>

<footer>
</footer>

</html>

Explicando:
Estrutura Básica

01 tag <!DOCTYPE html>

essa tag é utilizada sempre no inicio do documento para informar à máquina que nosso documento está em linguagem HTML

02 tag <html lang="pt-BR">

A tag html é uma tag básica para colocar todo o conteúdo html, mas a parte do lang="pt-br" serve para definir o tipo de linguagem daquele documento

03 tag <head></head>

Essa tag é a cabeça, tudo que está dentro dela não é mostrado para o usuário. Só vai ser lido pelo navegador ou o programa que está interpretando nossa linguagem/site

03.1 tag <meta>

Essa tag coloca algumas configurações padrões no navegador como: linguagem, compatibilidade com navegadores diferentes, tamanho padrão dos conteúdos etc... Falaremos disso melhor em outro momento.

03.2 tag "title"

Essa tag mostra o título do site. Aquele título que aparece na aba do navegador, no canto superior do monitor.

04 tag "header"

Tag que representa o cabeçalho do seu site

04.1 tag "nav"

Essa tag representa uma navegação, muito utilizada para criar menus, onde terá links que direciona para outro documento, página ou conteúdo.

05 tag "body"

Essa tag representa o corpo do conteúdo.

05.1 tag "h1"

Essa tag representa um título de nível 1. Mas teremos uma aula mais profunda sobre isso.

05.2 tag "p"

Essa tag representa um parágrafo. Mas teremos uma aula mais profunda sobre isso.

06 tag "footer"

aqui fica nosso rodapé.

02.1 /html

Todas as tags devem ser fechadas, salvo raras exceções, mas aqui no final é onde fechamos a tag html, aberta lá no inicio do documento.


Explicando:
Meta Tags

As Meta Tags estarão sempre na parte do "head" do nosso código.

As Meta Tags são dados e configurações iniciais que são lidas apenas pelos navegadores.

Definindo Linguagem

Fazendo um adendo, antes dos "head", abrimos uma tag "head" essa tag é fechada no final do nosso documento, depois do "footer". Dentro dessa tag, no inicio, nós cologamos a linguagem, representando pelo cod "land="linguagem""

Meta Charset UTF-8

Essa tag serve para definir o tipo de padronização e compatibilidade dos caracteres. No nosso caso, precisamos utilizara tag "charset="UTF-8"", permitindo ao navegador ler acentos, que são muito utilizados na lingua Portuguesa.

Meta XUACompatible e IEedge

Essa tag permite que alguns navegadores como Internet Explorer fiquem compatíveis com o seu site.

Meta ViewPort, Width=Device, Initial-Scale

Essas tags servem para configurar a área visivel do conteúdo adapatável para cada tamanho de dispositivo. Isso impede de acontecer por exemplo de usar o celular e abrir um navegador bem pequeno como se estivesse usando PC.

Meta Description

Mostra aquela sinópse do seu site quando é visto na busca do Google por exemplo.

Meta Keywords

Contém palavras chaves que facilitam os mecanismos de busca encontrar seu site.

Meta Author

É praticamente a assinatura do Programador, aqui os mecanismos de buscas também vai entendendo e guardando as páginas que você escreve, e acabam relacionando seu nome aos conteúdos.
