HTML

O html é a estrutura mais básica da página Web(o esqueleto da página), ele é responsável pelos textos em geral, imagens, audios, videos e a estrutura em geral.

Pode se encontrar um arquivo html no formato ".htm" mas é raramente usado hoje me dia. O formato do arquivo mais comum é ".html".

Hipertext

A expressão "HiperText refere-se aos links que conecta as páginas. Ou seja um documento que navega para outro documento ou página.

Markup Language

A expressão "Markup Language" refere-se sobre o tipo de linguagem do HTML. Muitos confundem HTML como linguagem de programação, mas o HTML não é linguagem de programação, é uma linguagem de marcação.

O que isso significa?

Bem, uma linguagem de marcação é feita de tags e rótulos, neste caso representadas pelo sinal de "< > Texto < / >"

Atualmente existem 140 tags HTML, algumas delas não são usadas com frequência e algumas já estão em desuso.
