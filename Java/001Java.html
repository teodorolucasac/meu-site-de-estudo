Case

No java podemos escrever em PascalCase, camelCase, minúsculas ou MAIÚSCULAS. Cada forma para determinada função.

PascalCase: representa Classe ou Interface

camelCase: representa Atributo, Variável ou Método

nome_minusculo: representa nome do Pacote

NOME_MAIUSCULO: representa uma Constante
