Funções

Uma função é um bloco de código projetado para executar uma tarefa específica. Como se fosse uma pequena linha de produção, onde vai entrar uma informação e vai sair outra.
Exemplo: uma esteira onde vai entrar madeira bruta e sair uma cadeira.

OBS: As funções no JS só são executadas quando são invocadas/chamadas.

function soma(valor1, valor2){
return valor1 + valor2;
}
document.getElementById("IdDoElemento").innerHTML = soma (10, 17)

function - palavra-chave para criar uma função.
soma - nome da função.
() - parenteses servem para colocar parâmetros, valores de entrada ou deixar sem valor também.
(valor1, valor2) - são 2 parâmetros, mas pode colocar quantos quiser.
return - é a palavra chave para criar a saída da função.
valor1 + valor2 - é o calculo que a saída vai transmitir.
var total = soma(10,17) - aqui foi a invocação/chamada da função. Neste caso o resultado vai aparecer no elemento com ID "IdDoElemento"

Também posso invocar de outras formas a função. Exemplo:
var total = soma (10,17);
alert(total);

Vamos fazer outra função

function realParaDolar(real, cotaDolar){
return real * cotaDolar;
}

var valorReal = 7.89;
var cotacao = 5.08;
var total = realParaDolar(valorReal, cotacao);
alert ("O valor em real é: R$"+valorReal+"o valor em dólar é: U$ "+ total)
Resultado: O valor em real é R$7.89 o valor em dólar é: U$40.08

01 Criamos a função: function
02 Chamamos ela de "realParaDolar"
03 Passamos 2 parâmetros, "real" e "cotaDolar"
04 Criamos a saída: return
05 Demos o cálculo/lógica da saída: real * cotaDolar
06 Criamos 2 variáveis "valorReal" e "cotacao"
07 Atribuimos valores fixos a elas "7.89" e "5.08"
08 Criamos a variável "total"
09 Atribuimos o valor da variável "total" como o retorno da função "realParaDolar"
10 Passamos os valores calculados na função atráves das variáveis do item 06
11 Invocamos/chamamos a função em um alerta
12 Resultado: O valor em real é R$7.89 o valor em dólar é: U$40.08

Essas funções também podem ser invocadas através botões e de eventos

function alertaHello() {
alert("Olá, Mundo!");
}
<button onclick="alertaHello()">Clique aqui!</button>


01 Criamos a função
02 Demos nome de "alertaHello"
03 Definimos a saída como a palavra-chave "alert"
04 Informamos qual é o valor do "alert" -> "Olá, Mundo!"
05 Criamos um botão
06 Configuramos o evento de click "onclick"
07 Chamamos/invocamos a função "alertaHellow" quando for acionado o evento de click no botão.

Outro exemplo

function paraCelsius(valorFahrenheit){
return (5/9) * (valorFahrenheit - 32);
}
var x - paraCelsius (77);

alert("A temperatura é de " + x + "graus celsius."");
Resultado: A temperatura é de 25 graus celsius.

Observações

01 Eu posso passar um valor da função como padrão mas ele pode receber outro.
Exemplo:
function multiplicando (n1=0, n2=0)
Assim se não for informado outro valor, o valor padrão será 0

02 Criar função no parâmetro de uma variável.
Exemplo:
let v = function (x){
return x*2
}
console.log(v(5))
Então eu chamei uma variável e ela executou uma função.
