Operadores

Operadores são alguns sinais que usamos no nosso código para executar alguma determinada função simples.

Os operadores podem ser separados em 6 categorias:

1. Aritméticos (matemáticos)
2. Atribuição
3. String (concatenação)
4. Comparação (relacionais)
5. Ternário (condicional)
6. Lógicos
7. Tipeof

01 Aritiméticos (matemáticos):

+ Soma
- Subtração
* Multiplicação
/ Divisão real
% Resto da divisão inteira
** Potência (elevado ao número X)
++ Incrimentação
-- Desincrementação

Precedência / Ordem de execução:
-> ()
-> **
-> *, /, %
-> +, -

02 Atribuição:

Os operadores de atribuição atribuem valores a variáveis JavaScript.

Operador Exemplo Igual a
= x = y x = y
+= x += y x = x + y
-= x -= y x = x - y
*= x *= y x = x * y
/= x /= y x = x / y
%= x %= y x = x % y
**= x **= y x = x ** y

03 String (concatenação)

O operador "+" também pode ser usado para adicionar (concatenar) strings.

let text1 = "John";
let text2 = "Doe";
let text3 = text1 + " " + text2;
Resultado: John Doe

O +=operador de atribuição também pode ser usado para adicionar (concatenar) strings:

let text1 = "What a very ";
text1 += "nice day";
Resultado: What a very nice day

Adicionando Strings e Números
Adicionar dois números retornará a soma, mas adicionar um número e uma string retornará uma string:

let x = 5 + 5;
let y = "5" + 5;
let z = "Hello" + 5;
Resultado de x , y e z:
10
55
Hello5

05 Ternário (condicional)

O operador ternário se chama ternário porque são compostos de 3 partes. São representados por 2 símbolos, interrogação "?" e dois pontos ":"

Exemplos:
01 - teste ? true : false
02 - média >= 7.0 ? "Aprovado" : "Reprovado"
03 - x % 2 == 0 ? "Par" : "Impar"

06 Lógicos

Os operadores lógicos é usando quando precisamos passar mais de uma condição para retornar um boolean.

&& e
|| ou
! não

Exemplo:
resultado = (idade > 60 && < 70);
