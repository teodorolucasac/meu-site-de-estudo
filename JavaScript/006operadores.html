Operadores

Operadores são alguns sinais que usamos no nosso código para executar alguma determinada função simples.

Os operadores podem ser separados em 6 categorias:

1. aritméticos (matemáticos)
2. atribuição
3. relacionais (de sequência)
4. comparação (typeof)
5. ternário (condicional)
6. lógicos

01 Aritiméticos (matemáticos):

+ Soma
- Subtração
* Multiplicação
/ Divisão real
% Resto da divisão inteira
** Potência (elevado ao número X)
++ Incrimentação
-- Desincrementação

Precedência / Ordem de execução:
-> ()
-> **
-> *, /, %
-> +, -
