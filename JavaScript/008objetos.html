Objeto

Objetos são basicamente variáveis com muitos valores dentro. Os valores dentro do objeto são chamados de propriedades.
Dentro dos objetos também pode ter métodos que é como se fosse uma função dentro de uma propriedade.

Exemplo:
Objeto: carro

Propriedades: carro.marca = ford
carro.modelo = ka
carro.cor = branco

Métodos: carro.partida()
carro.acelerar()
carro.freiar()

Criando objeto:

const carro = {marca:"ford", modelo:"ka", ano:2015, placa:"ABC-1234"};

carro é o objeto,
marca, modelo, ano e placa é propriedade,
ford, ka, 2015 e ABC-123 é o valor das propriedades.

É bem comum usar o const para criar objeto já que raramente iremos querer um valor variável nas propriedades do objeto.

Métodos no objeto:

const carro = {marca:"ford",
modelo:"ka",
ano:2015,
placa:"ABC-1234",
buzina: function() {alert("Biiiiiiiiiii!")}
completo: function() {
return "A marca é: "+this.marca+" e o modelo é: "+this.modelo};
};

console.log(carro); - vai mostrar o objeto com todos as propriedades e métodos
console.log(carro.ano); - vai mostrar o valor específico nessa propriedade específica
console.log(carro["marca"]); - outra forma de mostrar uma propriedade específica
carro.buzina(); - vai chamar/invocar o método do objeto
console.log(carro.completo()); - vai passar (A marca é: ford e o modelo é: ka)

Clonando Objetos

Para clonar objetos podemos usar 2 métodos.

Primeiro:
const carro2 = Object.assign({
portas: 5 portas
},carro);
dessa forma você pode adicionar propriedades como portas.

Segundo:
const carro3 = {...carro};
método mais simples e rápido, porém não é possível adicionar novas propriedades.
