Prototype

Os metodos de cada objeto no Javascript são herdados através do Prototype, ou seja cada tipo de dado diferente (typeof) recebe metodos e propriedades diferentes herdadas nativamente

Para acessar o tipo de dado (typeof) use:

    Object.getOwnPropertyNames(Array.prototype);

Para acessar o nome dos métodos herdados use:

    Object.getOwnPropertyNames(NodeList.prototype)
    Object.getOwnPropertyNames(HTMLCollection.prototype)
    Object.getOwnPropertyNames(Document.prototype)
