Saídas

saídas quer dizer a maneira que o JS tem para exibir uma informação ou um dado na tela

01 - innerHTML

O innerHTML é usado quando você quer imprimir / mostrar um conteúdo HTML na tela.

02 - document.write

Essa saída vai escrever algo no documento onde ela está localizada. Também responde quando usamos tags html e funções(mais usados para fazer testes)

03 - alert

Essa saída basicamente vai abrir um pop-up de alerta na página.

04 - console.log

Essa saída é mais discreta porque ela não vai mostrar algo na página em si, apenas vai mostrar a saída na aba console na area de desenvolvedor do navegador.
