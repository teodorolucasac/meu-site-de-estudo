Arrays

Arrays são variáveis onde você pode armazenar mais de um valor, mas diferente do objeto não pode passar nome a aos valores, ou melhor, os arrays não possuem propriedades. Se não pode passar nome, como é encontrado um valor de um array? Com base na posição que ele se encontra.

Exemplo array literal:
const lista = ["arroz", "feijão", "macarrão", "leite"]
Ou seja:
lista[0] vai ser arroz,
lista[1] vai ser feijão,
lista[2] vai ser macarrão,
lista[3] vai ser leite.

Outra forma de criar array:
const lista = []
lista[0] = "arroz";
lista[1] = "feijão";
lista[2] = "macarrão";
lista[3] = "leite";

Outra forma de criar array:
const lista = new Array ("arroz","feijão","macarrão","leite");

Obs: Os valores de um array podem ser substituidos igual um var
Exemplo:
lista[0] = "café";
Agora o arroz foi substituido pelo café

Mais funções de um Array

Apesar de um array não ter propriedades igual objetos, ele pode contar muitas outras coisas dentro dele como, um outro array, uma função e até mesmo um objeto

Diferenças do array e objeto

const pessoa = ["Lucas", "Teodoro", 30]; <- array
const pessoa = {nome:"Lucas", sobrenome:"Teodoro", idade:30}; <- objeto

pessoa.nome;
pessoa[0];
