JavaScript (JS)

JavaScript é a linguagem de programação usada para adicionar interatividade ao seu site.

O JavaScript é abreviado em JS, e seus arquivos são em formato ".js"

O JS estará sempre atrelado à um arquivo HTML e certamente junto a um CSS também. É o trio que compoem o desenvolvimento Web de forma perfeita.

JavaScript Interno

O JS interno é quando você coloca sua linha de código JS <script> </script> dentro do documento HTML.

Geralmente o código <script> </script> é colocado em baixo do fim da tag </body>. Isso porque o navegador carrega as linhas de cima para baixo e isso evita que o JS tente modificar algo que ainda nem foi carregado. Além disso a renderização do HTML e CSS será mais rápida melhorando a experiência do usuário, tendo em vista que o JS costuma ser a parte mais pesada do site.

JavaScript Externo

O JS Externo por sua vez é criado em um documento (.js) separado do documento HTML. Neste caso é necessário fazer uma referência no código HTML que importa o documento externo.js.
Exemplo:
<script src="../script.js">

</script>
