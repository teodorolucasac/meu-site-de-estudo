Datas

Para trabalhar com data, usaremos um objeto especial para datas do JS que é o Date, e sempre que criarmos uma data usaremos o new Date();

Comando base para pegar a data:
let data = new Date();
console.log(data);
resultado: vai informar a data e hora atual,
Thu Jul 07 2022 15:09:37 GMT-0300 (Horário Padrão de Brasília)

Comando para pegar o ano:
let ano = data.getFullYear();
console.log(ano);
resultado: 2022

Comando para pegar o mês:
let mes = data.getMonth();
console.log(mes);
resultado: 06
OBS: o detalhe é que o comando age como um objeto, então o mês começa do 0 com Janeiro e termina em 11 com Dezembro.

Para informar o mês por extenso precisamos fazer assim:
const mesesDoAno = ["Janeiro", "Fevereiro", "Março", "Abril","Maio", "Junho", "Julho", "Agosto", "Setembro", "Outubro", "Novembro", "Dezembro"];
let mesExtenso = mesesDoAno[data.getMonth()];
resultado: Julho

Comando para pegar o dia do mês:
let diaMes = data.getDate();
console.log(diaMes);
resultado: 7 (hj é dia 7)

Comando para pegar dia da semana:
let diaSemana = data.getDay();
console.log(diaSemana);
resultado: 4 (hj é quinta-feira)
OBS: aqui temos a mesma questão do mês, para o JS começamos a semana no numero 0 com o Domingo e terminamos no dia 6 com o Sábado

Para pegar o dia da semana por extenso:
const diaDaSemana = ["Domingo","Segunda","Terça","Quarta","Quinta","Sexta","Sábado"]
let diaSemanaExtenso = diaDaSemana[data.getDay()];
console.log(diaSemanaExtenso);
resultado: Quinta

Comando para pegar a hora:
let hora = data.getHours();
console.log(hora);
resultado: 15 (agr são 3hrs pm)

Comando para pegar os minutos:
let minutos = data.getMinutes();
console.log(minutos);
resultado: 31 (agr são 03:31 pm)

Comando para pegar os segundos:
let segundos = data.getSeconds();
console.log(segundos);
resultado: 58 (agr são 03:32:58 pm)

Comando para pegar os milissegundos:
let milisegundos = data.getMilliseconds();
console.log(milisegundos);
resultado: 879 (agr são 03:34:36.879 pm)

Comando para pegar a data no padrão brasileiro:
let dataBR = data.toLocaleString('pt-BR');
console.log(dataBR);
resultado: 07/07/2022 15:36:21

Comando para pegar data curta, sem horas, no padrão brasileiro:
let dataBRCurta = data.toLocaleString('pt-BR',{dateStyle: 'short'});
console.log(dataBRCurta);
resultado: 07/07/2022

Comando para pegar só as horas no padrão brasileiro:
let dataBRHoras = data.toLocaleString('pt-BR',{timeStyle: 'short'});
console.log(dataBRHoras);
resultado: 15:39

Outra forma de pegar a data no padrão BR é pegando todos os valores separados e concatenar:
d = new Date();
diaMes2 = d.getDate();
mes2 = d.getMonth();
ano2 = d.getFullYear();

let dataPadraoBR = diaMes2 + "/" + mes2 + "/" ano2;
console.log(dataPdraoBR);
resultado: 7/6/2022
OBS: aqui vamos encontrar 2 problemas
01 No dia menor que 10 ele não vai mostrar 2 casa numerais, ou seja dia 07 mostra 7,
02 No mês além de engolir o 0 ele vai mostrar 1 numero a menos por ser um objeto e começar da posição 0 e não da posição 1.

Para corrigir o problema acima:
d3 = new Date();
diaMes3 = d3.getDate();
mes3 = d3.getMonth() + 1;
ano3 = d3.getFullYear();

function addZero(x) {return x<10 ? '0'+x : ''+x};

let dataPadraoBR3 = addZero(diaMes3) + "/" + addZero(mes3) + "/" ano3;
console.log(dataPadraoBR3);
resultado: 07/06/2022
OBS: acrecentamos +1 na variavel mes3 e fizemos uma função que inclui o "0" antes do dia e mês.

Comando para comparar datas - Maior ou Menor:
var hoje = new Date();
var vencimento = new Date(2022, 0, 15);

if (hoje > vencimento){
console.log('Sua conta está vencida!');
} else {
console.log('Ainda não venceu, tudo certo!');
}
resultado: Sua conta está vencida!
OBS: no vencimento devemos informar ano, mes e dia. Porém no formato do JS, com janeiro começando em 0.

Comando para mostrar a diferença entre 2 datas:
var dataInicial = new Date();
var dataFinal = new Date(2022, 11, 31);
var diferencaTempo = dataFinal.getTime() - dataInicial.getTime();
var diferencaDias = Math.ceil(diferencaTempo / (24 * 60 * 60 * 1000));
console.log(diferencaDias);
resultado: 177 (faltam 177 dias para o ultimo dia do ano)
OBS:
01 getTime é para pegar os milisegundos e sempre que vamos fazer calculo com data pegaremos os milisegundos.
02 (diferencaTempo / (24 * 60 * 60 * 1000) este é o calculo para converter os milisegundos em dias.
