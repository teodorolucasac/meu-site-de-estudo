Constructors

Constructor também conhecido como classes no JS são basicamente usados para criar objetos. Pode se dizer então que classes/constructor são funções especiais para criar objetos.

Exemplo sintaxe:
class Carro{
constructor(valor1, valor2, valor3){
this.marca = valor1;
this.modelo = valor2;
this.ano = valor3;
}
buzina(){
return this.modelo+" buzinou: Biiiii!";
}
}

const uno = new Carro('Fiat','Uno','2001');
const gol = new Carro('Valkswagen','Gol','2013');

OBS: quando vamos nomear classes sempre usaremos Pascal Case, ou seja, começaremos com a primeira letra maiúsculas
As declarações de classes são consideradas não içadas, ou seja não tem como criar um objeto por classes digitando o código acima do constructor.

Neste caso criamos um constructor com 3 propriedades, marca, modelo e ano, e uma função de buzina
