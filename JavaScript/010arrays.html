Arrays (matrizes)

Arrays são variáveis onde você pode armazenar mais de um valor, mas diferente do objeto não pode passar nome a aos valores, ou melhor, os arrays não possuem propriedades. Se não pode passar nome, como é encontrado um valor de um array? Com base na posição que ele se encontra.

Exemplo array literal:
const lista = ["arroz", "feijão", "macarrão", "leite"]
Ou seja:
lista[0] vai ser arroz,
lista[1] vai ser feijão,
lista[2] vai ser macarrão,
lista[3] vai ser leite.

Outra forma de criar array:
const lista = []
lista[0] = "arroz";
lista[1] = "feijão";
lista[2] = "macarrão";
lista[3] = "leite";

Outra forma de criar array:
const lista = new Array ("arroz","feijão","macarrão","leite");

Obs: Os valores de um array podem ser substituidos igual um var
Exemplo:
lista[0] = "café";
Agora o arroz foi substituido pelo café

Mais funções de um Array

Apesar de um array não ter propriedades igual objetos, ele pode contar muitas outras coisas dentro dele como, um outro array, uma função e até mesmo um objeto

Diferenças do array e objeto

const pessoa = ["Lucas", "Teodoro", 30]; - array
const pessoa = {nome:"Lucas", sobrenome:"Teodoro", idade:30}; - objeto

Para selecionar valores:
pessoa.nome;
pessoa[0];

Métodos

pessoa.length; length retorna quantas valores possui o array
pessoa[pessoa.length -1]; vai retornar o ultimo item do Array
Array.isArray(pessoa); vai retornar se é um array (true) ou não (false).
pessoa.join(" - "); vai colocar - entre cada valor.
pessoa.push("Brasileiro"); vai acrescentar um valor a última posição do array
pessoa.shift(); vai acrescentar um valor na primeira posição do Array.
pessoa.pop(); vai remover o último valor do Array.
pessoa.shift(); vai remover o primeiro valor do Array.
delete pessoa[0]; vai excluir o valor informado, mas vai deixar a posição undefined, ou seja, não vai alterar as posições dos demais.
pessoa.splice(1,0, "adicionar1", "adicionar2"); splice adiciona o item na posição informada, depois informa se quer deletar algum valor da direita, depois informa o valor que vai adicionar.

const lista1 = ["arroz","feijão","leite",macarrão];
const lista2 = ["suco","refrigerante","carne"];

const superLista = lista1.concat(lista2); vai concatenar 2 arrays.
const meiaLista1 = superLista.slice(3); vai pegar somente os valores apartir da posição 3 do array superLista.
const meiaLista2 = superLista.slice(3, 5); só vai pegar da posição 3 ao 5.
superLista.sort(); vai ordernar em ordem alfabetica. Se for números ele só vai contar a primeira casa numeral.
superLista.reverse(); vai inverter a ordem que estava anteriormente.

const numeros = [40,100,1,5,25,10];

numeros.indexOf(25); vai procurar a posição do valor 25 do array.
numeros.sort(function(a,b){return a - b}); dessa forma vai colocar todos os números na ordem crescente correta;
numeros.sort(function(a,b){return b - a}); dessa forma vai colocar todos os números na ordem decrescente correta;

function maiorNumero(array){
return Math.max.apply(null,array);
}
maiorNumero(numeros); vai pegar o maior número do array (tem que fazer a função).

function menorNumero(array){
return Math.min.apply(null,array);
}
menorNumero(numeros); vai pegar o menor número do array (tem que fazer a função).


function filtragem(value, index, array){
return value >20;
}
const maior20 = numeros.filter(filtragem);
console.log(maior20); vai mostrar somente números maiores que 20.
