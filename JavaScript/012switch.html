Switch

Switch, assim como if e else, também é um tipo de declaração comparativa. O switch é utilizado para ter diferentes ações com base em diferentes condições no mesmo bloco de verificação.

Exemplo sintaxe:
switch (expressão){
case a:
//ação que vai acontecer
break;
case b:
//ação que vai acontecer
break;
case c:
//ação que vai acontecer
break;
default:
//ação que vai acontecer
}
