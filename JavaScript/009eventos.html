Eventos

Eventos é uma das se não a parte mais importante para entender e dominar no JS.

Eventos HTML são "coisas" que acontecem com elementos HTML. Quando o JavaScript é usado em páginas HTML, o JavaScript pode "reagir" a esses eventos.

Um evento HTML pode ser algo que o navegador faz ou algo que um usuário faz.
Aqui estão alguns exemplos de eventos HTML:

1- Uma página da Web HTML terminou de carregar
2- Um campo de entrada HTML foi alterado
3- Um botão HTML foi clicado

Eventos mais comuns:

onclick -> dispara quando recebe click;
ondblclick -> dispara com clique duplo;
onmouseover -> dispara quando o mouse está sobre;
onmouseout -> dispara quando o mouse é movido para fora;
onmousemove -> dispara quando o mouse é movido no elemento;
onmousedown -> dispara quando o clique do botão foi pressionado;
onmouseup -> dispara quando o clique do botão é liberado;
onfocus -> dispara quando o elemento recebe foco. Valido para input...;
onchange -> dispara quando exite uma mudança no conteúdo "ao mudar";
onblur -> dispara quando o elemento perde o foco;
onkeydown -> dispara quando uma tecla é pressionada e solta;
onkeypress -> dispara quando uma tecla é solta sobre um elemento;
onkeyup -> dispara quando uma tecla é solta sobre um elemento;
onload -> dispara quando uma página termonou o carregamento. Body;
onresize -> dispara quando há um redimencionamento da janela.
