Variáveis

As variáveis são como recipientes onde podemos armazenar informações que podem varias.

Imagine um recipiente de vidro onde você pode colocar dentro dele qualquer coisa. canetas, moedas, bomboms...
Eu posso chamar esse recipiente como eu quiser. Exemplo Pote

Exemplo:
var Pote="Bomboms"
var é uma declaração para criar a variável;
Pote é o nome do recipiente;
Bomboms é o valor atribuido (o conteúdo do recipiente);

Nome = Identificador

O nome de cada variável é o IDENTIFICADOR
Regras dos Identificadores
1. PODE começar com LETRA
2. PODE começar com $
3. PODE começar com _
4. PODE usar letras seguidas de números
5. PODE usar acentos e símbolos
6. NÃO podem começar com NÚMEROS
7. NÃO pode colocar espaço (substitui por _)
8. NÃO pode ser palavras reservadas (palavras que o JS usa como comandos)
9. Maiúsculas e Minúsculas fazem diferença
Dica: Crie nomes coerentes das suas variáveis.

Funcionalidade da Variável

Exemplo:
var a=2;
var b=3;
var c=a+b;

Ou seja, em um campo de entrada de uma página, o usuário pode digitar qualquer número, mesmo que o número vá variar, o resultado ficará correto porque a saída sempre será a + b independente do valor.

Outra maneira de criar variáveis

var a,b,c;
a=2;
b=3;
c=a+b;
console.log(c)
Resultado: 5

Valor Indefinido

var d;
console.log(d)
Resultado: undefined
O recípiente foi criado mas o valor não foi definido.

Valor Not a Number (NaN)

var d,e,f;
f=d+e
console.log(f)
Resultado: NaN
Somar 2 variáveis indefinido a máquina não consegue entender que é um número e devolve NaN.

Concatenação

Concaternar é basicamente juntar varias variáveis em uma só linha ou texto.
Exemplo:

nome="Lucas";
sobrenome="Teodoro";
nomeCompleto="nome+" "+sobrenome";

document.getElementById("idDoElemento").innerHTML=nomeCompleto;
Resultado: Lucas Teodoro

Concaternar números e letras

var nome, sobrenome, nomeCompleto, idade; nome="Lucas";
sobrenome="Teodoro";
idade=27;
pessoa=idade+10+nome;

document.getElementById("idDoElemento").innerHTML=pessoa;
Resultado: 37Lucas
OBS: Foi somado os número e concatenado com as letras.


pessoa=nome+idade+10;
document.getElementById("idDoElemento").innerHTML=pessoa;
Resultado: Lucas2710
OBS: Foi concatenado tudo, tanto letras quanto números. Isso porque começou com palavras e foi interpretado tudo como palavras (strings).

O ideal é não misturar strings e numbers na concatenação, caso queira concaternar os 2 coloque o número em uma variável.

Diferença entra var, let e const

No JS nós temos 3 palavras-chaves para declarar uma variável:
-> var
-> let
-> const

01 - let e const não permite ser redeclarado. Diferente do var.

02 - o var é uma variável coringa, ela vai funcionar (e vazar) em todo o código, dentro e fora de um escopo/bloco (blocos são representados pelas chaves {}).
03 - Já o let não vaza para fora nem para dentro do escopo/bloco, mas pode ser redefinido em valores diferentes dentro de cada escopo/bloco.
04 - O const por sua vez é uma variável constante, ela nunca vai mudar, nunca pode ser redefinida independente se está fora ou dentro de um escopo/bloco.
