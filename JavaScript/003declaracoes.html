O que é declaração

Um programa é basicamente feito de instruções, dados e informações que o computador consegue interpretar. Na linguagem de programação essas instruções chama-se declarações. Ou seja, um programa ou um código JS completo é formado por uma lista de declarações interpretada pelo navegador.

Essas declarações podem ser composta de várias coisas como valores, operadores, expressões, funções, comentarios etc...

É comum em um documento JS ter centenas e as vezes milhares de linhas de comandos/declarações. O que separa as declarações é o sinal de ponto e virgula;

Algumas Regras

No JavaScript, as declarações inteiras podem ser escritas sequencialmente na mesma linha ou quebrando linha, não importa, o JS não faz distinção.

Também não tem problema quebrar uma única declaração em outras linhas, porém é ideal deixa-las em ordem para acostumar com o padrão das demais linguagens.

O JS também permite fazer a declaração sem espaço entre os comandos ou com espaço, tanto faz, mas algumas linguagens deve ser escritas sem espaço.

o JS também tem uma série de comandos pré prontos de fábrica dele, como break, continua, debugger, do, for, function, if, else, return, twitch, var etc... Iremos conhecer todas elas.
