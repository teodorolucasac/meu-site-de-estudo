Overflow

A propriedade overflow especifica quando o conteúdo de um elemento de nível de bloco deve ser cortado, exibido com barras de rolagem ou se transborda do elemento.

overflow: visible;

Visible é o valor padrão do overflow de qualquer elemento. Caso o elemento não tenha especificado o seu tamanho, ele vai adaptar conforme o conteúdo. No entando se o elemento estiver tamanho definido o conteúdo vai ser exibido mesmo se ultrapassar o tamanho definido ao elemento

overflow: scroll;

O valor scroll vai basicamente criar uma barra de scroll na lateral para navegar e visualizar o conteúdo excedente do tamanho do elemento. Lembrando que a barra de scroll vai ser visível mesmo que o conteúdo não ultrapasse o tamamnho do elemento para evitar de ficar aparecendo e desaparecendo com conteúdos dinâmicos

overflow: auto;

O valor auto vai fazer o conteúdo excedente criar uma barra de rolagem scroll, porém quando o conteúdo não for excedente ele vai desaparecer com a barra de rolagem.

overflow: hidden;

O valor hidden vai básicamente esconder/sumir com o conteúdo excedente, não mostrando nem criando barra de rolagem.

Efeitos de texto com Overflow

Neste caso usamos um white-space para o texto não quebrar linha, depois overflow para esconder o conteúdo excedente do container, depois text-overflow para decidir o que acontece com o texto excedido, neste caso adicionamos reticências "...". Por fim colocamos um :hover com white-space: normal;
.titulo{
white-space: nowrap;
overflow: hidden;
text-overflow: ellipsis;
}

.caixa:hover .titulo{
white-space: normal;
}
