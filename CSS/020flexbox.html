Flexbox

A propriedade Flexbox provê ferramentas para criação rápida de layouts complexos e flexíveis, e características que se mostraram historicamente difíceis com CSS.

Por um longo tempo, as únicas ferramentas compatíveis entre browsers disponíveis para criação de layouts CSS eram coisas como floats e posicionamento. Estas são boas e funcionam, mas em alguns casos também são limitadas e frustrantes. Os requisitos de layouts a seguir são difíceis ou impossíveis de se conseguir com estas duas ferramentas:

  • Centralizar um bloco de conteúdo verticalmente dentro de seu pai.
  • Fazer com que os filhos de um container ocupe uma quantidade igual de largura/altura disponível, independente da quantidade de largura/altura disponível.
  • Fazer todas as colunas de um layout com múltiplas colunas adotem a mesma altura, mesmo que contenham uma quantidade diferente de conteúdo.

Por sua vez, o flexbox faz muitas tarefas de layouts de maneira mais fácil. Vamos nos aprofundar!

Termos do Flexbox

O main axis é o eixo que corre na direção em que os flex items estão dispostos (por exemplo, as linhas da página, ou colunas abaixo da página.) O início e o fim do eixo é chamado main start e main end.

O cross axis é o eixo perpendicular (interceptador) que corre na direção em que os flex items são dispostos. O início e o fim deste eixo são chamados de cross start e cross end.

O elemento pai que possui display: flex configurado é chamado de flex container.

Os itens iniciados como flexible boxes dentro do flex container são chamados flex items.

flex-direction: ; (coluna ou linha)

flex-direction especifica a direção do eixo principal / main axis (em qual direção os filhos da flexbox estarão arranjados)
Exemplo: flex-direction: column;

Por padrão seu valor é row (linha) o que significa que os itens estarão alinhando um ao lado do outro. O valor column(coluna) vai alinhar os itens como um bloco, tomando toda a área como uma coluna única.

flex-wrap: ;

flex-wrap vem com o valor padrão "nowrap", ou seja, sempre que o conteúdo atingir as margens da tela ele continuará criando uma barra de rolagem. Já o valor "wrap" vai fazer com que o conteúdo quebre a linha para continuar sem criar barra de rolagem.
Exemplo sintaxe:
flex-wrap: wrap;

flex-flow: ;

O flex-wrap e flex-direction podem fazer um shorthand utilizando a propriedade flex-flow: ;
Exemplo sintaxe:
flex-flow: row wrap;

Alinhamento flexbox

justify-content: flex-start;

O justify-content é uma propriedade que vai permitir o alinhamento dos elementos em row / linha / horizontal.

Essa propriedade vem com o valor padrão :flex-start;, mas possui varios valores para se trabalhar como:
:left; esquerda
:right; direita
:center; centro
:space-around; espaçado em volta (espaço igual em torno dos elementos)
:space-between; espaçado entre elementos (entre main start e main end)
Obs: sempre na horizontal.

align-items: stretch;

A propriedade align-items vai permitir o alinhamento dos elementos flex em column / coluna / vertical.

Essa propriedade vem com o valor padrão :stretch;, mas possui os valores:
:stretch; esticado (do cross start ao cross end)
:flex-end; ao final do pai
:flex-start; no inicio do pai
:center; no centro do pai
Obs: sempre na vertical.

align-content: ;

A propriedade align-content desrespeita ao alinhamento de todo o conteúdo junto, diferente do align-items que trata do conteúdo individualmente.

Essa propriedade vem com o valor padrão :space-between;, mas possui outros valores:
:center; centraliza
:flex-end; ao final do pai
:flex-start; no inicio do pai
:space-around; margens iguais em torno
:space-between; margens iguais entre os itens
Obs: sempre na vertical.
