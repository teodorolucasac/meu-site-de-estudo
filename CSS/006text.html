Text

A propriedade Text é bem simples, mas podemos ver muitos atributos e configurações envolvendo os Textos.

Propriedades

text-align: center;
Alinha a posição do texto
(valores: center, end, justify, left, right, start, inherit, initial, unset)

text-decoration: none;
Altera decorações padrões de texto, como o underline dos links
(valores: overline, line-through, underline)

text-transform: uppercase;
Altera a letra para maiuscula, minuscula etc...
(valores: uppercase, lowercase, captalize)

text-indent: 10px;
Recua a primeira linha, fazendo efeito de parágrafo

letter-spacing: 10px;
Altera o espaçamento entre as letras

line-height: 0.5;
Altera espaçamento entre linhas

word-spacing: 15px;
Altera o espalamento entre as palavras

direction: rtl;
Inverte a posição do texto

unicode-bidi: bidi-override;
Inverte a posição das letras

text-shadow: 5px 5px 10px gray;
Cria uma sombra no texto
