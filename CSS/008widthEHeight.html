Width

A <width> define a largura de um elemento. Por padrão, ele define a largura da área de conteúdo , mas se <box-sizing> estiver definido como <border-box>, ele define a largura da área da borda.

As propriedades min-width e max-width substituem width.

Exemplo:
.a1{
background-color: aqua;
width: 300px;
height: 100px;
text-align: center;
}


Sintaxe

/* <length> values */
width: 300px;
width: 25em;

/* <percentage> value */
width: 75%;

/* Keyword values */
width: max-content;
width: min-content;
width: fit-content(20em);
width: auto;

/* Global values */
width: inherit;
width: initial;
width: revert;
width: revert-layer;
width: unset;

Valores

<length>
Define a largura como um valor absoluto.

<percentage>
Define a largura como uma porcentagem da largura do bloco que o contém.

<auto>
O navegador irá calcular e selecionar uma largura para o
elemento especificado.

<max-content>
A largura intrínseca preferida.

<min-content>
A largura mínima intrínseca.

<fit-content>(<length-percentage>)
Usa a fórmula fit-content com o espaço disponível substituído pelo argumento especificado, ou seja:
min(max-content, max (min-content, <length-percentage>)).

Height

Todas as específicações aplicadas no <width> também se aplica no <height> porém, agora estamos falando de altura e não largura como no width.
