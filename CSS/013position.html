Position

A propriedade position, define como um elemento pode ser posicionado (renderizado) no documento (página).
Existem 5 tipos de valores para a propriedade CSS position.
Veja:

position: static;
position: relative;
position: fixed;
position: absolute;
position: sticky;

OBS: o eixo central de um elemento vem padronizado como left/top, levando em conta a margem também. Caso queira alterar a posição do eixo use a propriedade: (transform: translate(-50%, -50%);)

Position: static;

O valor static é o valor padrão dos elementos e se comporta em uma camada, não podendo avançar em cima da position de outros elementos.

Position: relative;

O valor relative libera camada do eixo-Z (z-index: 0;), ou seja, coloca o elemento em camadas afrente ou atrás de outros elementos. E por isso trabalha o position do elemento ignorando a position (top, right, bottom, left)de outros elementos

z-index: 0;

O z-index é uma propriedade que altera ou define a camada de um elemento, podendo ser alterada apenas se o position for relative.

Position: fixed;

O valor fixed vai definir uma posição fixa de um determinado elemento com base na tela do usuário. Portanto não importa a rolagem da tela o elemento vai ficar preso naquela posição em relação a tela.

Position: absolute;

O valor absolute define a posição de um elemento com base ao parent dele (ancestral) no qual ele está contiudo.
Porém o valor absolute só vai conseguir basear no elemento Pai se este elemento tiver qualquer valor de position que não seja static, caso contrário ele vai continuar procurando outro elemento Pai até encontrar um elemento final como body, ou o header por exemplo.

Position: sticky;

O valor sticky vai grudar o elemento na tela, caso a tela comece a sair das margens e posição definida. Muito usado para menus, caixa de loguin ou cabeçalho de tabelas para acompanhar a rolagem para baixo por exemplo.
