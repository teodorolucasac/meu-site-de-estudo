Font

Font é a estilização dos textos. Temos diversas propriedades para estilizar as fonts, veja:

Font-Family

Font-family é a propriedade que altera a família das fonts.
Podemos escolher a família por um nome de família ou um nome de família genérica.
Exemplo:

/* Aqui temos o nome de família de fontes e nome de família genérica */
font-family: "Gill Sans Extrabold", sans-serif;
font-family: "Goudy Bookletter 1911", sans-serif;

/* Aqui temos apenas o nome de família genérica */
font-family: serif;
font-family: sans-serif;
font-family: monospace;
font-family: cursive, handwriting;
font-family: fantasy, display;

Podemos colocar o valor de varias famílias de font no <font-family> para evitar a incompatibilidade com alguns navegadores, além disso é um habito saudável colocar um nome de família genérica no final do valor.

Também é uma boa prática citar com aspas os nomes de família de fontes que contenham espaços em branco, dígitos ou caracteres de pontuação que não sejam hífens.

Mais detalhes sobre <font-family> no site da mozilla.

Font-Size

Essa é a propriedade que define o tamanho da font.

/* Valores Absolutos */
cm: centímentro
mm: milímetro
in: polegada
px: pixel
pt: ponto
pc: paica

/* Valores Relativos */
em: tamanho atual da font
ex: altura X de uma font
rem: altura X de uma font do root (configurado no body)
vw: largura da View Width
vh: altura da View Height
%: porcentagem

OBS: Recomendação da W3C é sempre usar PX ou EM, evitar outras unidades de medida de tamanho da font.
Para saber mais sobre as propriedades e valores, confira no site do mozilla.

Font-Style

A propriedade font-style define se uma fonte deve ser estilizada com uma face normal, itálica ou oblíqua de sua família de fontes.
Exemplo:

font-style: normal; (estilo padrão)
font-style: italic; (estilo itálico)
font-style: oblique; (estilo semelhante ao itálico)
font-style: oblique 10deg; (semelhante ao itálico podendo mudar o angulo)

Font-Weight

A propriedade CSS font-weight define o peso (ou negrito) da fonte. Os pesos disponíveis dependem da família de fontes que está definida no momento.
Exemplo:

/* Keyword values */
font-weight: light;
font-weight: normal;
font-weight: bold;

/* Keyword values relative to the parent */
font-weight: lighter;
font-weight: bolder;

/* Numeric keyword values */
font-weight: 100;
font-weight: 200;
font-weight: 300;
font-weight: 400;// normal
font-weight: 500;
font-weight: 600;
font-weight: 700;// bold
font-weight: 800;
font-weight: 900;

Font Externa

Para adicionar uma font externa é preciso fazer um "@import" no CSS, informando URL e propriedades da font (esse import tem que ser nas primeiras linhas do seu CSS).
Exemplo:

@import url('https://fonts.googleapis.com/css2?family=Roboto:ital,wght@0,100;0,300;0,400;0,500;0,700;0,900;1,100;1,300;1,400;1,500;1,700;1,900&display=swap');

Depois é só usar a propriedade font-family normal com o nome da fonte, este que está em negrito no exemplo (nome apos a palavra "family=" do link).
Exemplo:

font-family: 'Roboto', sans-serif;

Font Interna

Para inserir uma font interna, ou seja, uma font que está hospedada na sua máquina, é preciso primeiro baixa-la e coloca-la na devida pasta. Depois faremos um import da sua pasta.
Exemplo:

@font-face {
font-family: 'nome da familia';
src: url('local do arquivo') format('tipo do arquivo');
}

Obs: para cada valor da font (size, style, etc) você precisará fazer um import diferente pois será um arquivo de font diferente.

font-family: ' ' ;: deve colocar o nome que desejar entre aspas.
src: url(' ') local exato das pastas até o arquivo.
format(' '): Os principais tipos de arquivo de font são:
Tipos de format()
- opentype (otf)
- truetype (ttf)
- embedded-opentype
- truetype-aat (Apple Advanced Typography)
- svg
