Propriedade Color

A propriedade Color é basicamente utilizada para alterar a cor da letra/font.
(color: #ffffff;).

Padrão de cores

As máquinas não entende a cores como nós pelo visual normalmente, mas existem alguns padrões para identificação das cores pela máquina.

Valor Padrão

background-color: darkblue;

HEX

Exemplo: #0088ff

RGB

Exemplo: rgb(0, 136, 255)

HSL

Exemplo: hsl(208, 100%, 50%)

CMYK

Exemplo: 100, 47, 0, 0

Obs: que o valor deve ser um solid color. Não pode ser um , que na verdade é um tipo de .
