Estudo sobre Seletores

Os seletores são a identificação do elemento HTML que o CSS quer modificar.

OBS: o nome das atribuições class ou id nao podem começar com um valor númerico.


Voltar

Seletor Global

Os seletores globais usam as Tags para configurar o estilo, dessa forma todas as Tags do seu site vai ser configurada.
Exemplo:

html {

}

body {

}

Seletor por Classe

Os seletores por classe é um seletor que vai aplicar o estilo apenas se você atribuiu determinada classe para determinada tag. É possível usar a mesma classe para várias tags. As classes são aplicadas com o simbolo de ".".
Exemplo:

.classeX {

}

.classeY {

}

Seletor por Identificador

O seletor por indentificador usa uma atribuição de id para aplicar um estilo. O ID é uma atribuição única, não sendo possível atribuir o mesmo id para duas ou mais tags. As classes são aplicadas com o simbolo de "#".
Exemplo:

#paragrafo1 {

}

Seletor Universal

O seletor universal é representado pelo simbolo "*", o que for configurado nele vai alterar todo o seu documento, desde o primeiro Parent até o último Child.
Exemplo:

*{

}

OBS: o Seletor Universal está abaixo da hierarquia de um Seletor direto

Chamando um Seletor

Podemos selecionar mais de um Seletor na mesma configuração, basta colocar uma "," entre eles
Exemplo:

h1, h2, p {

}

Para selecionar os parents e childs, basta coloca-los um na frente do outro em ordem ou usando o simbolo ">".
Exemplo:

body section p {

}

OU

body > section > p {

}

Hierarquia de Seletores

Via de regra:
01 Quanto maior o parent do Seletor, maior é na hierarquia, perdendo apenas para o último child.
02 O último child é mais forte que qualquer parent sozinho (obs: considera-se ultimo child o id, antes dele a class e só depois a tag).
03 O seletor parent + child sucessores é mais forte que qualquer parent ou child sozinho.
