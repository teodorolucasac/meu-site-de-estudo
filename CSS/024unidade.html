Font-Size

Essa é a propriedade que define o tamanho da font.

/* Valores Absolutos */
cm: centímentro
mm: milímetro
in: polegada
px: pixel
pt: ponto
pc: paica

/* Valores Relativos */
em: relativo ao tamanho atual da font
ex: altura X de uma font
rem: relativo ao tamanho atual do root (configurado do html)
vw: largura da View Width (0-100)
vh: altura da View Height (0-100)
%: porcentagem do elemento pai

OBS: Recomendação da W3C é sempre usar PX, EM ou REM, evitar outras unidades de medida de tamanho da font.

Mas por padrão iremos usar o REM, para caso alguma alteração no tamanho da font do elemento pai não altere o tamanho da font do elemento filho.

Para saber mais sobre as propriedades e valores, confira no link:
https://developer.mozilla.org/en-US/docs/Web/CSS/font-size

View Height e View Width

Diferença de % e vw/vh: Quando usamos vw/vh o tamanho da nossa tela será a referência. Quando usamos % o tamanho do elemento pai será a referência.

Ou seja, se usarmos 100% de um elemento que o pai tenha 100vw realmente vamos atingir o 100vw. Mas se o pai tiver outro valor menor não vamos atingir a tela toda, vamos atingir apenas 100% do elemento pai.

Quando usamos width: 100vw; e o elemento está com margin ou padding, o elemento vai estourar na tela.

Função calc()

Essa função de CSS permite fazer calculo de medidas, desta forma podemos inclusive misturar unidade. Exemplo:

div{
width: calc(100vw / 3);
}
obs: vai pegar 100vw e dividir por 3 (33,33%)

misturando medidas
div{
width: calc(100vw - 100px);
}
obs: independente do tamanho da tela vai retirar 100px
