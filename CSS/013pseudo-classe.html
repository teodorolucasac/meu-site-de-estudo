:Pseudo-classes

Pseudo-classe é basicamente definir a propriedade com base em um estado de um determinado elemento.
As pseudo-classes são definidas com o simplo ":"
Veja um exemplo de sintaxe:

seletor:pseudo-classe {
propriedade: valor;
}

Alguns exemplos de pseudo-classe são:
:visited historico de navegação
:checked status do conteúdo
:hover posição do mouse
:focus Elemento em foco, usando o teclado (tab)
:active Quando clicamos no elemento

Seletores

:first-child

Seleciona o primeiro elemento.

:last-child

Seleciona o último elemento.

:nth-child(4)

4 (quarto elemento), even (pares), odd (ímpares), 3n (de 3 em 3).

Exemplo:

        .lista-1 li:first-child{
            background-color: #caf;
        } 
seleciona o primeiro elemento

    
        .lista-1 li:last-child{
            background-color: #cfa;
        } 
seleciona o ultimo elemento

    
        .lista-1 li:nth-child(3){
            background-color: #fca;
        } 
seleciona o elemento indicado na função

    
        .lista-2 li:nth-child(even){ /*even=par e odd=impar*/
            background-color: #fac;
        } 
seleciona todos os elementos pares ou impares

    
        .lista-2 li:nth-child(3n){
            background-color: #acf;
        } 
seleciona todos os elementos pulando o valor da função
  1. Primeiro elemento
  2. Segundo elemento
  3. Terceiro elemento
  4. Quarto elemento
  5. Quinto elemento

Também podemos ignorar elementos da seguinte forma:

        /* todos menos first-child e last-child */
        ol li:not(:first-child, :last-child) {
        background: #adf;
        }

        /* todos h2's dentro de section, menos o que tiver a classe .contato */
        section h2:not(.contato) {
        text-decoration: underline;
}
