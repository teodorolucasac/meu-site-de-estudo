Pripriedade Background-image

O background-image serve para preencher o fundo de um elemento com uma determinada imagem

Atributos mais usados em background-image:
background-image: url(imagem/01background.jpg); insere uma imagem.
background-size: contain; ajusta a imagem no fundo.
background-repeat: no-repeat; faz a imagem repetir para algum lado.
background-position: center; ajusta o ponto de fixação da imagem

Shorhand

Shorhands são usados para fazer uma configuração com vários atributos em uma linha apenas. Neste caso podemos fazer assim.

Exemplo:
background: #00ffff url(imagem/01background.jpg)
no-repeat center;
background-size: contain;
Obs: nesse caso o size não entra no shorhand.

Background Degradê

Também é possível fazer um background em degradê usando o atributo background-image e passando o valor linear-gradient.

Exemplo:
background-image: linear-gradient(to right, #0088ff, #00ffff 40%);
Nesse caso estamos passando o valor de:
linear-gradient(direção do degradê, primeiraCor, segundaCor %deOcupaçãoNaTela)

Também podemos usar graus para informar a direção do degradê
Exemplo:
background-image: linear-gradient(90deg, #0088ff, #00ffff 40%);

Também é possível fazer o grandient de forma circular, basta passar o valor radial no lugar de linear.
Exemplo:
background-image: radial-gradient(#0088ff, #00ffff);
