Margin

A propriedade margin do CSS define a área de margem nos quatro lados do elemento.
A tag <margin> é uma abreviação que define todas as margens individuais de uma só vez:

Diferente do padding, a <margin> preenche a parte de fora do elemento.

Atributos

O margin possui atributos indicando a direção dos 4 lados do elemento.
Exemplo:
padding-bottom: 10px;
padding-left: 50%;
padding-right: 2em;
padding-top: auto;

Shorhands

/* Aplica para todos os quatro lados */
margin: 1em;

/* vertical | horizontal */
margin: 5% auto;

/* topo | horizontal | inferior */
margin: 1em auto 2em;

/* topo | direita | inferior | esquerda */
margin: 2px 1em 0 auto;

/* Valores globais */
margin: inherit;
margin: initial;
margin: unset;

Obs: A propriedade margin pode ser especificada usando um, dois, três ou quatro valores. Cada valor deve ser um <length>, uma <percentage>, ou a palavra-chave <auto>. Cada valor pode ser positivo, zero ou negativo.

Valores

length
O tamanho da margem como um valor fixo.

percentage
O tamanho da margem como um percetual, relativo à largura do bloco em que o elemento está contido.

auto
O navegador seleciona uma margem adequada para utilizar. Por exemplo, em alguns casos este valor pode ser utilizado para centralizar o elemento.
