Responsivo

Um layout responsivo se adapta a diferentes tipos/tamanhos de tela e as preferências do usuário. (tema light/dark por exemplo)

Características:

        Tamanhos
        Unidades relativas como %, vw, fr são mais utilizadas.

        Media Queries
        Código será ativado de acordo com o tamanho da tela.
        Alterar o layout, esconder itens e adicionar funcionalidades.

        Flexbox e Grid
        Manipulamos o flexbox e o grid para adaptar o conteúdo.

        Tipo de tela
        Temos que levar em consideração a operação do site apenas com o dedo.

meta viewport

A meta tag viewport é essencial para garantir com que o HTML/CSS lide de forma correta com telas de alta densidade. Ela normaliza o PX do css.

   <meta name="viewport" content="width=device-width, initial-scale=1">

name="viewport" nome da tag
content="width=device-width ajusta o tamanho do conteúdo = a largura do dispositivo
initial-scale=1" escala de zoom, "1" para padrão

Media Queries

Media Queries (consultas de mídia) é um módulo de CSS que permite ajustar visualização do conteúdo a uma variedade de dispositivos/telas.

A @media gera um bloco de código que só será ativado caso a condição entre parênteses seja verdadeira.

OBS: A @media respeita a hierarquia das folhass de estilo, ou seja, o conteúdo abaixo no código tem prioridades nos de cima.

Diminua/Aumente o tamanho da tela
para mudar de cor

Sintaxe:

Exemplo 1: após 600px aplique fundo vermelho
@media only screen and (min-width: 600px) {
div{
background-color: red;
color: black;
}
}

Exemplo 2: após 600px até 768px, aplique fundo vermelho
@media screen and (max-width: 768px) and (min-width: 600px) {
div{
background-color: red;
color: black;
}
}

Exemplo 2: após 1000px ou abaixo de 5000, aplique fundo vermelho
@media screen and (min-width: 1000px), (max-width: 500px) {
div{
background-color: red;
color: black;
}
}

@media screen

Essa propriedade vai aplicar um valor na tela do usuário. Mas caso ele tente imprimir a tela por exemplo, este valor não vai sair.

@media print

Essa propriedade vai aplicar um valor quando o usuário for imprimir uma página. Caso ele só navegue normalmente não vai aparecer nada.

Responsivo com Grid repeat(auto-fit)

        .minmax {
            grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
          }
          
        .galeria {
            grid-template-columns: repeat(auto-fit, 150px);
          }

auto-fit vai criar o máximo de colunas respeitando o tamanho informado.

O valor informado só vai funcionar corretamente se for um valor fixo ou min ou max ou minmax.

Object Fit

object-fit essa propriedade vai fazer com que uma imagem que tenha um height fixo não distorça.

object-position vai referenciar onde será o corte da imagem.

Exemplo:

        .grid {
            display: grid;
            grid-template-columns: 1fr 1fr;
            gap: 1rem;
          }
          
          .cover img {
            height: 100%;
            object-fit: cover;
            object-position: top center;
          }

max-width

max-width

O max-width determina um valor máximo de largura do elemento. Não deixe os elementos livres na tela, principal os textos.

ch

Unidade relacionada à largura do caractere 0 da tipografia.

Largura do texto

Controlar o tamanho máximo da largura do texto é essencial para garantir uma boa leiturabilidade. Para corpo de texto entre 50-75 caracteres é o ponto ideal.

margin: 0 auto;

margin top e bottom 0. margin left e right auto (automática, define valores iguais com base no espaço em branco e alinha o conteúdo ao centro).

prefers-color-scheme

A @media prefers-color-scheme irá executar o código css conforme a preferência de tema do usuário. Funciona da mesma forma que o @media querie com o max-width, porém agora em relação ao modo escuro (dark) ou claro (light)

            @media (prefers-color-scheme: dark) {
                :root {
                  --fundo: #222;
                  --texto-1: #eee;
                  --texto-2: #bbb;
                }
              }
              
              @media (prefers-color-scheme: light) {
                :root {
                  --fundo: #fff;
                  --texto-1: #111;
                  --texto-2: #444;
                }
              }
