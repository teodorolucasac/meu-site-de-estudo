Padding

A propriedade <paddin> define a área de preenchimento em todos os quatro lados de um elemento de uma só vez.

Diferente do margin, o <padding> preenche a parte de dentro do elemento.

Exemplo com <padding>

Exemplo com <margin>

Ou seja, o background também acompanha.

Atributos

Assim como o margin, o <paddin> também possui atributos indicando as 4 direções do elemento.
Exemplo:
padding-bottom
padding-left
padding-right
padding-top

Sintaxe

/* Apply to all four sides */
padding: 1em;

/* vertical | horizontal */
padding: 5% 10%;

/* top | horizontal | bottom */
padding: 1em 2em 2em;

/* top | right | bottom | left */
padding: 5px 1em 0 2em;

/* Global values */
padding: inherit;
padding: initial;
padding: revert;
padding: revert-layer;
padding: unset;

A propriedade <paddin> pode ser especificada usando um, dois, três ou quatro valores. Cada valor é um ou um . Os valores negativos são inválidos.
