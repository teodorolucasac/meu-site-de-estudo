Pseudo-Elemento

Os mais importantes e mais usados pseudo-elements são o ::before e ::after. Eles criam elementos HTML com base no seletor.

Uso prático: São utilizados para decorarmos o conteúdo, com eles evitamos o uso de elementos desnecessários no HTML.

Valores

Todo Pseudo-Elemento precisa de um valor content e um valor de display para funcionar corretamente. Para criar um valor vazio basta usar as aspas. content: " ";

Veja o exemplo:

  h1::after {
    content: '';
    display: block;
    width: 40px;
    height: 4px;
    background: #caf;
  }
  
  li {
    font-size: 1.5rem;
    display: flex;
    align-items: center;
  }
  
  li::before {
    content: '';
    display: inline-block;
    width: 8px;
    height: 8px;
    border-radius: 50%;
    background-image: linear-gradient(#caf, #fff);
    border: 2px solid #caf;
    margin-right: 10px;
  }
