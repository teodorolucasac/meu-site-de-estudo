Media Queries

Media Queries (consultas de mídia) são um módulo de CSS que permite ajustar visualizações a uma variedade específica de dispositivos/telas sem mudar o conteúdo.

Diminua/Almente o tamanho da tela
para mudar de cor

Sintaxe:

Exemplo 1:
@media only screen and (min-width: 600px) {
.cor{
background-color: red;
color: black;
}
}

Exemplo 2:
@media screen and (max-width: 768px) and (min-width: 600px) {
.cor{
background-color: red;
color: black;
}
}
