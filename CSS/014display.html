Display

Display é uma das propriedades mais importantes do CSS quando se trata de controlar o layout. Ela especifica o tipo de caixa de um elemento e, SE ou COMO esse elemento vai ser exibido.

Cada elemento HTML já tem um valor de display padrão de acordo. Se ele vai ocupar a width toda,ou se ele vai ficar um do lado do outro...

Tipos de display

display: block; ocupa toda a width
(ex: div, h1~6, p)
display: inline; alinhas uma ao lado da outra
(ex: a, span, img)
display: inline-block; alinhas uma ao lado da outra
(nenhum elemento pré configurado com inline-block)
display: none; desaparece com o elemento
(muito usado para criar eventos de aparecer e desaparecer)

Display Block

O display block é basicamente um display que vai ocupar a largura (width) completa da página. Além disso, ele também vai obedecer todos as definições de largura e tamanho.

Display Inline

Como dito, o display inline alinha seus elementos de forma horizontal (lado a lado).

O inline comporta como se cada elemento fosse uma Palavra. Se eu der espaço no código ele vai criar um espaço como se fosse um texto, se eu não der espaço ele ficará colado.
Exemplo:

  • Primeiro item
  • Segundo item
  • Terceiro item
  • Primeiro item
  • Segundo item
  • Terceiro item

O display inline, diferente do block não obedece propriedades que alteram os valores de altura (height) e largura (width). Ele vai obedecer apenas o tamanho do objeto que compoem

Inline-Block

O display inline-block é uma configuração que vai se comportar em geral como o display inline, posicionando seus elementos horizontalmente, porém vai respeitar as definições de altura e largura como no display block.
Exemplo:

  • Primeiro item
  • Segundo item
  • Terceiro item
