CSS

O CSS é a ferramente que define o estilo, as cores, fontes do site. Em outras palavras, é o altera a estética do site.

O CSS faz um casamento perfeito com HTML, pois o HTML vai cuidar do conteúdo e o CSS do estilo.

CSS significa: Cascading Style Sheets, traduzindo ao Português: Folha de Estilo em Cascata

O arquivo CSS possui um formato ".css"

CSS Inline

CSS inline é quando eu passo código de estilo dentro de um Tag HTML. Em outras palavras, eu coloco um código de estilo dentro da linha de código HTML.

O CSS inline só vai funcionar exclusivamente para aquela Tag expecífica na qual ele foi inserido.
Exemplo:

<p style="background-color: yellow; color: blue; font-size: 25px;">Lorem...</p>

CSS Interno

O CSS interno é inserido no inicio do documento HTML, mais precisamente antes do fechamento da tag </head>. Neste caso o CSS vai funcionar para todo aquele documento, se eu editar um parágrafos, todos os meus parágrafos também podem receber aquela configuração de estilo.
Exemplo uma edição de todos os parágrafos de uma class exemplo:

.exemplo p {
background-color: red;
color: #7df2f8;
}

CSS Externo

O CSS externo é uma documento de CSS separado do documento HTML, sendo que dento do arquivo HTML teremos um link direcionando/referenciando aquele arquivo externo CSS. Neste caso eu posso colocar vários arquivos HTML, ou seja, várias páginas de HTML direcionando para o mesmo arquivo CSS, fazendo com que páginas diferentes recebam a mesma configuração de estilo

Para configurarmos o CSS externo é preciso criar um arquivo CSS independente no formato ".css"
Exemplo: style.css
E dentro do arquivo HTML / dentro da tag <head> eu faço um link para o arquivo CSS
Exemplo: <link rel="stylesheet" href="../style.css">

Pronto, agora qualquer arquivo HTML que tiver o link para seu arquivo CSS receberá as devidas configurações de estilo.
