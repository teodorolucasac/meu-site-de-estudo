Display Grid

Vamos aprender a dividir o contúdo em colunas com Display Grid

Voltar

display: grid;

Grid é um formato de display que trabalha os Elementos em forma de colunas (columns). Quando colocamos o display grid no Elemento pai, mudamos completamente o comportamento dos filhos e também do pai em relação aos filhos.

Um elemento pai podem ter filhos com outros tipos de display ou em grid também, sem problemas.

grid-template-columns

Essa propriedade define quantos grid terá seu Elemento pai e qual o tamanho desse grid.

fr

fr é uma unidade de medida muito utilizada para definir o tamanho das columns do seu grid. Tendo em vista que, as unidades de medida em porcentagem como "vw" e o próprio "%" ignora as margins fazendo criar barras de rolagem na página. Já o "fr" age como se fosse "%" mas utilizada apenas o resto do espaço depois das margens.

gap

gap é uma propriedade utilizada para criar distância entre as grids, como se fossem margens. Lembrando que no gap a unidade fixa como "px" vão funcionar mais corretamente.

align-content

Propriedade que alinha na vertical

justify-content

Propriedade que alinha na horizontal

place-content

Propriedade que faz um shorhand para configurar o align e justify, nesta ordem.

Valores:
start; alinha do início
center; alinha do centro
end; alinha do final
stretch; completa o alinhamento com padding
space-between; cria espaços iguais entre elementos mas não nas margens
space-around; cria espaço iguais entre os elementos e as margens
space-evenly; tenta distribuir mais igualmente os espaços entre elementos e as margens

grid-column: 1/3;

Quando queremos colocar cada elemento em uma coluna específica, usamos a propriedade grid-column: ;

Sendo certo que, o valor um número isolado significa qual coluna o elemento vai ficar. Exemplo: grid-column: 3; vai ficar na coluna 3

Caso queira colocar em mais de uma coluna basta usar o "/" e passar o número das LINHAS, e não mais colunas. Exemplo: grid-column: 1/3; vai da linha 1 até a linha 3.

O valor -1 significa ultima linha. Exemplo grid-column: 1/-1; este elemento vai da linha 1 até a última, independente do número de linhas.

Se quisermos expandir passando o valor de colunas precisamos usar o valor de span. Exemplo: grid-column: span 3; vai expandir 3 colunas a partir da coluna que o elemento nasceu.

Outro exemplo de span é. Exemplo: grid-column: 1 / span 3; vai começar na coluna 1 e expandir 3 colunas.

align-items e justify-items

Propriedade do Elemento Pai que alinha os itens dentro da coluna, ou seja de todos os itens juntos.

align-items: alinhamento vertical
justify-items: alinhamento horizontal
place-items: shorthand para vertical/horizontal

align-self e justify-self

Propriedade do Elemento Filho, e alinha os itens individualmente

align-self: alinhamento vertical
justify-self: alinhamento horizontal
place-self: shorthand para vertical/horizontal

grid-template-rows

grid-template-rows: 100px auto;
Define o tamanho das linhas.

grid-auto-rows:
Linhas são adicionadas automaticamente com o valor de auto.

grid-row: 2
Define a linha do item, funciona da mesma forma que o grid-column.
